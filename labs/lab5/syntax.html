1. Подсветка синтаксиса

Модуль обеспечивает динамическую подсветку синтаксиса для языков программирования (Python, JavaScript, C++ и др.).

Для пользователя: Откройте файл с кодом, и подсветка автоматически применится. Перейдите в "Настройки > Темы" для смены цветовой схемы.

Пример: Напишите def hello(): print("Hi!") в Python — ключевые слова выделятся цветом.
