Быстрый старт

Установка

  1. Скачайте установочный файл с нашего сайта
  2. Запустите установщик и следуйте инструкциям
  3. После установки запустите программу

Первая настройка

При первом запуске вам будет предложено:

  • Выбрать цветовую тему (светлая/темная)
  • Указать предпочитаемые языки программирования
  • Настроить горячие клавиши

Создание первого файла

  1. Нажмите Файл → Новый (Ctrl+N)
  2. Выберите тип файла (например, Python)
  3. Начните писать код - подсветка синтаксиса включится автоматически
