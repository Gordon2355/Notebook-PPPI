Интеграция с системами контроля версий

Встроенная поддержка Git и других VCS позволяет работать с версионностью прямо из редактора.

Основные возможности

  • Просмотр изменений в реальном времени
  • Создание коммитов без терминала
  • Визуальное разрешение конфликтов
  • Просмотр истории файлов

Работа с Git

  1. Откройте панель "Контроль версий" (Ctrl+Shift+G)
  2. Инициализируйте репозиторий для проекта
  3. Делайте коммиты через интуитивный интерфейс
  4. Пушите изменения на удаленный репозиторий
