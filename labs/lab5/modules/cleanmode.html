Режим "Чистого кода"

Минималистичный интерфейс для максимальной концентрации на коде.

Особенности режима

  • Скрытие всех панелей инструментов
  • Фокусировка на текущем файле
  • Темный фон с подсветкой только активного кода
  • Минимальные отвлекающие элементы

Как активировать

Используйте:

  • Горячую клавишу F11
  • Меню: Вид → Режим чистого кода
  • Автоматически при бездействии (настраивается)

Настройки

Вы можете настроить:

  1. Уровень затемнения неактивных частей кода
  2. Показ номеров строк
  3. Отображение текущей позиции в файле
