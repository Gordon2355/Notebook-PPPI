Модульные плагины расширений

Расширяйте функциональность редактора с помощью системы плагинов.

Установка плагинов

  1. Откройте Плагины → Магазин плагинов
  2. Найдите нужный плагин через поиск
  3. Нажмите "Установить"
  4. Перезагрузите редактор при необходимости

Популярные плагины

Плагин Описание
Code Metrics Анализ сложности кода
Remote Development Работа с удаленными серверами
AI Assistant ИИ-помощник для программирования

Создание своих плагинов

Используйте наш API для разработки:

class MyPlugin extends PishiPlugin {
  constructor() {
    super("My Cool Plugin");
  }
  
  onActivate() {
    console.log("Plugin activated!");
  }
}
