Подсветка синтаксиса

Модуль подсветки синтаксиса автоматически распознает языки программирования и выделяет ключевые элементы кода разными цветами.

Поддерживаемые языки

  • Python
  • JavaScript/TypeScript
  • HTML/CSS
  • C/C++/C#
  • Java
  • PHP
  • И многие другие...

Настройка цветов

Для изменения цветовой схемы:

  1. Откройте Настройки → Подсветка синтаксиса
  2. Выберите элемент для настройки (ключевые слова, строки, комментарии и т.д.)
  3. Укажите желаемый цвет и начертание
  4. Нажмите "Применить"

Определение языка

Программа определяет язык по:

  • Расширению файла (.py, .js, .html и т.д.)
  • Первым строкам содержимого (например, shebang в Unix-системах)
  • Явному указанию пользователя
