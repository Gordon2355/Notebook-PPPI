Линтинг и анализ кода

Встроенная система линтинга помогает находить ошибки и потенциальные проблемы в коде до его выполнения.

Поддерживаемые линтеры

  • ESLint (JavaScript)
  • Pylint (Python)
  • PHP_CodeSniffer (PHP)
  • Checkstyle (Java)
  • И другие...

Типы обнаруживаемых проблем

Тип Пример
Синтаксические ошибки Отсутствующая скобка, точка с запятой
Стилевые проблемы Несоответствие code style
Потенциальные баги Использование необъявленной переменной
Уязвимости безопасности SQL-инъекции, XSS

Настройки

Вы можете:

  1. Включить/отключить конкретные правила
  2. Установить уровень серьезности (ошибка, предупреждение, информация)
  3. Добавить пользовательские правила
