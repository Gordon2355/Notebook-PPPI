Инструментарий "Живого выполнения"

Запускайте и тестируйте код прямо в редакторе без переключения между окнами.

Поддерживаемые языки

  • Python (с поддержкой Jupyter-ноутбуков)
  • JavaScript (Node.js и браузер)
  • PHP (встроенный сервер)
  • Bash/Shell

Как использовать

  1. Откройте файл с кодом
  2. Нажмите F5 или кнопку "Выполнить"
  3. Просмотрите результат во встроенной консоли
  4. Используйте точки останова для отладки

Встроенный терминал

Доступны:

  • Несколько сессий терминала
  • Поддержка PowerShell, CMD, Bash
  • Быстрые команды для сборки проекта
  • Интеграция с системными утилитами
