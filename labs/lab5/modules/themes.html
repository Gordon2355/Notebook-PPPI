Создание своей темы

  1. Откройте Настройки → Темы → Редактор тем
  2. Выберите базовую тему для наследования
  3. Настройте цвета, шрифты и отступы
  4. Сохраните тему под новым именем

Автоматическое переключение

Настройте автоматическое переключение между светлой и темной темой:

  • По времени суток (дневная/ночная)
  • По системным настройкам ОС
  • По освещенности (если есть датчик света)
